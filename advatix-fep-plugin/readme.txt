== Description ==

Advatix FEP API Connection is used for synchronization woocommerce store with Advatix FEP.

It sync your woocommerce products, inventory.
It sync your woocommerce orders and manage order statuses.
It will create new order statuses.

**THIRD PARTY INFO**

This plugin uses <a href="https://developer.advatix.net/">Advatix.net</a> developer services which helps integrate your woocommerce products and orders to sync in multi-directional.
After generating your API credentials from developer.advatix.net, it starts synchronization of products and orders into your woocommerce store and from your woocommerce store to <a href="https://developer.advatix.net/">Advatix FEP</a>.

= How can I use the plugin? =
